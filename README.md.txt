# My Advanced Blog

This is a simple and responsive To-Do List web application built using **HTML**, **Bootstrap**, **CSS**, and **JavaScript**. It allows users to manage their daily tasks easily by adding, marking as completed, deleting, and filtering them.

## Features

- Add new tasks  
- Delete tasks  
- Mark tasks as completed  
- Filter tasks by:
  - All
  - Active
  - Completed  
- Tasks are saved in localStorage so they remain even after reloading the page
- Fully responsive design using Bootstrap 4

## Tools & Technologies

- HTML5
- CSS3
- Bootstrap 4
- JavaScript (Vanilla)
- Font Awesome (for icons)

## Setup and Usage

1. **Clone the repository**
2. **Open the project**
- Open `index.html` in your browser to view the site
- For development, use the unmagnified CSS version in `css/style.css`
- For production, the minified version `css/style.min.css` is referenced in the HTML
